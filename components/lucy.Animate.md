# lucy.Animate

Updates the children on each frame by calling their `update` function.

Typical usage:

```ascii
[ three.WebGLRenderer ]
[ lucy.Animate ]
[ three.Scene ]
[ three.Mesh ]
```

## Provide

* time: [lucy.Time](../types/lucy.Time.ts)
