# three.WebGLRenderer

Transforms the screen into a WebGL animation using [three.js](http://threejs.org).

Typical usage:

```ascii
[ three.WebGLRenderer ]
[ lucy.Animate ]
[ three.Scene ]
[ three.Mesh ]
```

## Provide

* object3D: [THREE.Scene](http://threejs.org/docs/#Reference/Objects/Scene) as `THREE.Object3D`
